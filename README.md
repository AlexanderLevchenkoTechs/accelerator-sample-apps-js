###Configuration

Add a `config.json` file with your OpenTok credentials to the `src` directory:
```javascript
{
  "apiKey": "YOUR_API_KEY",
  "sessionId": "YOUR_SESSION_ID",
  "token": "YOUR_TOKEN"
}
```


To run:
```javascript
$ npm install
$ npm run start
```
